Week 02 - Javascript: Novice to Ninja

Programming Basics

Variables have to be declared before they can be used. From ES6 onwards, JavaScript uses the keywords const and let to declare variables. The keyword const is used when the variable will not be reassigned to another value, whereas let is used if the variable might be reassigned later in the program.

Arrays, Logic and Loops

In JavaScript we have data structures such as arrays, sets, and maps. This chapter also included topics based on the different types of logical statements in JavaScript that allow the control flow of the program.

Arrays

An array is an ordered list of values. Here's an example of how to create an array on JavaScript:

const myArray = ['eggs', 'milk', 'butter'];

Adding Values to Arrays

myArray.push('juice');

Removing Values from Arrays

delete myArray[3];

Destructuring Arrays

Destructuring an array is the concept of taking values out of an array and presenting them as individual values.

const [x,y] = [1,2];

Merging Arrays

The concat() method can be used to merge an array with one or more arrays:

myArray.concat(['spice', 'salt']);

The join() Method

A method that can be used to turn an array into a string that includes all items in an array. The values are seperated by commas.

myArray.join();

Sets

Sets were introduced in ES6, as a data structure that represents a collection of unique values to avoid having duplicate values.
Here's an example of how you can create a set:

const list = new Set();
