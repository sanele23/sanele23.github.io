CSS3 Transforms and Transitions

Transforms

The CSS3 transform property allows you to translate, rotate, scale, and/or skew any element on the page. Transform functions can manipulate an element's appearance.

Scaling

The scale(x,y) function scales an element by the defined factors horizontally then vertically. If only one value is provided, it will be used for both the x and y values, growing or shrinking your element or pseudo-element while maintaining the original aspect ratio.



Canvas, SVG, and Drag and Drop

Canvas

An HTML5 API that enables drawing of anything through JavaScript. This will improve the performance of a website by avoiding the need to download images off the network. "With canvas, we can draw shapes and lines, arcs and text, gradients and patterns." Canvas also gives us the power to manipulate pixels in videos and images.

Drag and Drop

An API that can be used to add some fun user interaction to a website. The API allows you to specify that certain elements are draggable, and then specify what should happen when these draggable elements are dragged over or dropped onto other elements on the page.
