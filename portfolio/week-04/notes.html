Forms

In html, you create a form by using the "form" element. Inside the form, you can get controls such as, input fields, select menus and buttons. Forms are used when you want to record data and save it to a server/database.

The form is named search, and includes two controls such as the input field and button. The input field is to enter the word you're searching for and the button is to submit the search input of the form. The action attribute is the URL the form will be submitted to so it can be processed on the server.

Accessing Form Elements

Forms trigger a number of events, and some of these events are exclusive to forms. The focus events occurs when an element is focused on.

Form Controls

Forms have different types of controls to make the webpage more interactive. The following is the different types of controls in forms:

  • input fields, including text, passwords, check boxes, radio buttons, and file uploads
  • select menus for drop-down lists of options
  • textarea elements for longer text entry
  • button elements for submitting and resetting forms
